Job Board App

This project is a full-stack Python, React, and SQL application. 
It was originally intended as a simple to-do list app to practice core development concepts across the stack. 
On March 1, 2026, I repurposed it into a personal job board application to better align with my current goals.

The app allows me to store and manage job opportunities by saving:
-Job title
-Job posting link
-Priority rating

This provides a centralized view of roles I’m interested in and helps reinforce practical backend and frontend development skills.

Tech Stack
Backend: Python
Frontend: React
Database: SQL

Backend Structure
The Python backend is organized for clarity and separation of concerns:
main.py – Application entry point and core execution
utils.py – Helper functions and shared logic
models.py – Data and response models

Purpose
This project represents the evolution of a learning exercise into a practical tool. 
It allows me to continue practicing CRUD operations, database integration, API structure, and frontend state management 
while building something directly useful for tracking job opportunities.
